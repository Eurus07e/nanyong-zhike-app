# Nanyong Zhike v1.1.7 Release Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Publish v1.1.7 with a deterministic Simplified Chinese Windows installer and a fast Windows preflight on ordinary CI and before native release compilation.

**Architecture:** The repository vendors a version-compatible translation from an immutable `jrsoftware/issrc` commit. A shared PowerShell script checks its SHA-256 and compiles the real Inno script around a disposable placeholder executable. Both CI and Release call this script; the final Release step compiles the real installer and asserts the exact EXE exists.

**Tech Stack:** PowerShell 7, Inno Setup 6.7.1, GitHub Actions, pytest, Python, React/Vite.
