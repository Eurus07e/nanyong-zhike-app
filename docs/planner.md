# 我的计划

“我的计划”是南雍知课内置的本地学习看板。用户可以从本学期课表勾选课程，补充目标、手动课程和资料，使用以下模板创建计划：

- 学期稳分：适合日常课程、考试与周任务管理；
- 跨专业准入课：适合转专业、辅修或跨院系课程准备。

计划支持任务勾选、周复盘、多计划切换、删除以及 JSON 导入导出。导入文件最大为 1 MB，资源链接只接受 `http:` 和 `https:`，不会执行导入文件中的脚本、HTML 或表达式。

## 数据位置与账号隔离

MVP 不新增后端写接口。计划保存在当前浏览器的 `localStorage`，存储键按当前登录账号生成不可逆的短哈希，以避免同一浏览器上不同账号直接共用活动计划。该隔离用于避免误展示，不属于加密，也不能替代操作系统账号隔离。

计划不会保存密码、认证 Cookie、CASTGC、完整成绩响应或完整课表响应。只保存用户主动加入的课程号、课程名称、学分、目标、任务、资料和复盘。

清理浏览器数据会导致计划丢失。用户应使用“导出当前计划”生成 JSON 备份；公共电脑使用完毕后，可以在计划页面删除当前计划或清空本机全部计划。

## 当前边界

- 不支持服务器同步和跨设备自动同步；
- 不支持上传 PDF、任意 JavaScript 模板或远程 Markdown 抓取；
- 模板内容是非官方的计划辅助，不判断转专业资格、课程抵免或选课结果；
- 首版从当前课表读取课程，培养方案中的“加入计划”入口留待后续单独实施。


## Windows 双击启动

在源码目录根部双击 `启动南雍知课.cmd`。首次启动会自动创建 `.venv`、安装 Python 依赖、构建前端，并连接同级原发行包中的 `nju-cli.exe`；之后每次双击会重新构建最新页面并启动本地服务，浏览器会自动打开。

源码文件夹 `nanyong-zhike-app` 需要与原发行包文件夹 `NanyongZhike-windows-x86_64` 保持在同一父目录。关闭启动窗口或在窗口中按 `Ctrl+C` 即可停止服务。
