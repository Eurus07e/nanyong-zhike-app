南雍知课 v1.0 使用说明
=======================

不需要安装 Python、Node.js 或 Docker。

1. 完整解压下载的压缩包。
2. macOS 双击“启动南雍知课.command”。
   Windows 双击“启动南雍知课.cmd”。
   Linux 双击“启动南雍知课.sh”。
3. 等待默认浏览器自动打开。
4. 关闭启动窗口即可停止南雍知课。

程序只监听 127.0.0.1，不会向局域网或公网开放。

首次运行提示未知开发者时，请先确认文件来自：
https://github.com/Eurus07e/nanyong-zhike-app/releases

macOS 可按住 Control 点击启动文件并选择“打开”；Windows 可在 SmartScreen 中选择
“更多信息”，确认来源后再运行。

本地数据位置：
macOS   ~/Library/Application Support/NanyongZhike
Windows %LOCALAPPDATA%\NanyongZhike
Linux   ~/.local/share/NanyongZhike

排名与平均学分绩来自仅支持 HTTP 的南京大学交换生旧系统，该查询不具备 HTTPS 传输保护。

详细说明、隐私策略和故障排查请查看同目录 README.md。
