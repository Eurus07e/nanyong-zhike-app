# 安全说明

南雍知课会处理南京大学统一身份认证和个人学业数据。请不要在公开 Issue、截图、日志或复现
文件中包含真实密码、CASTGC、Cookie、学号、成绩或课表。

发现安全问题时，请通过 `yuxuanshu@smail.nju.edu.cn` 私下联系维护者，并提供不含真实凭据的
最小复现步骤。维护者确认问题前，请不要公开漏洞细节。

仅仓库默认分支的最新版本接受安全修复。生产部署必须使用 HTTPS、随机 `APP_SECRET`，并限制
`.env` 与持久化数据卷的访问权限。

排名与平均学分绩来自目前只支持 HTTP 的南京大学交换生系统。本地发行包为保持功能完整会
按需访问该系统；公网部署默认关闭此链路。该上游连接无法提供 HTTPS 的机密性与完整性保护，
不应把 `ALLOW_INSECURE_EXCHANGE_SYSTEM=true` 用于不受信任的远程部署环境。
